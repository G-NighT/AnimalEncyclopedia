Руководство пользователю

Запуск приложения производится с рабочего стола.
После загрузки приложения на экране отображается главное окно программы для выбора дальнейшего действия с имеющимися кнопками меню для получения справки и выхода из программы, кнопками навигации по программе в разделах: «Таблицы», «Запросы», «Отчёты», где к таблицам относятся кнопки: «Тип / Класс / Отряд» и «Животные и их среды обитания», к запросам: «Длина жизни от “?” до “?”», «Перекрёстный запрос», «Редактор SQL-кода», к отчётам: «Длина жизни от “?” до “?”», «Среда обитания животных», «Типы животных».
Главное окно программы представлено на рисунке 6.

Рисунок 6 – Главное окно программы

При нажатии на кнопку «Тип / Класс / Отряд» появляется форма для отображения, выбора и редактирования данных о типах, классах и отрядах животных. Для этого следует в таблицах «Тип», «Класс» и «Отряд» определить необходимый данные животного.
Окно «Тип/Класс/Отряд» представлено в приложение C на рисунке C.1.
При нажатии на кнопку «Животные и их среды обитания» в окне «Тип / Класс / Отряд», либо в главном окне программы, осуществляется переход на окно «Животные и их среды обитания», где также нажатием на кнопку «Тип / Класс / Отряд» осуществляется переход на окно «Тип / Класс / Отряд» для выбора другого типа, класса или отряда.
Само окно «Животные и их среды обитания» служит для добавления новых животных в уже выбранный ранее отряд, либо редактирования уже имеющихся, а также для добавления характерных им сред обитания, либо редактирования уже имеющихся.
Помимо этого, в случае, если количество животных будет превышать число, когда пользователь способен самостоятельно обнаружить необходимое животное, предусмотрена возможность быстрого поиска их по нажатию мышкой на таблице «Животные» и, на клавиатуре, на начальные буквы искомого животного.
Окно «Животные и их среды обитания» представлено в приложении C на рисунке C.2.
При нажатии на кнопку «Добавить» под таблицей «Животные» появляется форма для добавления нового животного в определённый ранее отряд, где также существует возможность смены отряда животного нажатием на кнопку «Выбрать», а название животного и характеристика – текстовые поля, изображение животного – гиперссылка, а длина жизни – числовое поле.
Окно «Добавление нового животного» представлено на рисунке 7.
 
Рисунок 7 – Окно «Добавление нового животного»

По нажатию кнопки «Сохранить и выйти» записи помещаются в базу данных и форма для добавления нового животного закрывается, но, в случае, если какое-либо из полей было не заполнено, программа даёт об этом знать.
Окно «Оповещение о пустом поле» представлено на рисунке 8.
 
Рисунок 8 – Окно «Оповещение о пустом поле»
Аналогичное действие происходит по нажатию на кнопку «Добавить ещё», только форма для добавления нового животного не закрывается, а принимает первичное состояние, где все поля пусты. При нажатии на кнопку «Отмена» форма для добавления нового животного закрывается.
При нажатии на кнопку «Добавить» под таблицей «Среда обитания» появляется форма для добавления среды обитания для выбранного ранее животного, где название среды обитания и климат среды обитания – текстовые поля, а площадь, млн км2 – числовое поле. 
При нажатии кнопок «Сохранить и выйти», «Добавить ещё» и «Отмена» происходят аналогичные действия, как и для формы «Добавление нового животного».
Окно «Добавление среды обитания» представлено на рисунке 9.
 
Рисунок 9 – Окно «Добавление среды обитания»

При нажатии на кнопку «Длина жизни от “?” до “?”» из раздела «Запросы» на главном окне программы, появляется форма для отображения запроса с указанием значений длины жизни, ограниченных интервалами для поля «От» от 0 до 99 и для поля «До» от 1 до 999. И при нажатии на кнопку «Сформировать SQL-запрос» формируется SQL-запрос согласный этим значениям.
Окно «Запрос «Длина жизни от “?” до “?”»» представлено в приложении C на рисунке C.3.
При нажатии на кнопку «Перекрёстный запрос животных» из раздела «Запросы» на главном окне программы, на весь экран появляется форма для отображения перекрёстного запроса животных.
Окно «Перекрёстный запрос животных» представлено в приложении C на рисунке C.4.
При нажатии на кнопку «Длина жизни от “?” до “?”» из раздела «Отчёты» на главном окне программы, появляется окно с отображением отчёта согласно запросу «Длина жизни от “?” до “?”» предоставляющее возможность печати сконструированного отчёта, сохранения его в файл и т.п.
Аналогичные действия происходят при нажатии на кнопки: «Среда обитания животных» и «Типы животных», меняются лишь содержания отчётов соответственно темам, по которым они конструируются.
Окна отчётов представлены в приложении C на рисунках C.5 – C.7.

3.5 Руководство пользователю-разработчику

При нажатии на кнопку «Редактор SQL-кода» из раздела «Запросы» на главном окне программы, появляется форма для отображения редактора SQL-кода. Эта форма создана на тот случай, если пользователю будет необходимо самостоятельно осуществить выборку по собственному запросу заполнив поле SQL-редактора и нажав на кнопку «Выполнить», в случае, если поле SQL-редактора пусто программа оповещает пользователя аналогично рисунку 8.
Присутствует возможность загружать SQL-код из текстовых файлов с расширением .txt и сохранять в файлы с таким же расширением, по нажатию кнопок «Загрузить» и «Сохранить» соответственно. При нажатии на кнопку «Очистить» поле SQL-редактора очищается, а при нажатии на кнопку «Вернутся» окно закрывается и открывается главное окно программы.
Окно «Редактор SQL-кода» представлено в приложении C на рисунке C.8.
